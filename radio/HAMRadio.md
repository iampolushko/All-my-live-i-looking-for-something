# Правовая база

СЕРТ («European Conference of Postal and Telecommunications administrations«) — это Европейская конференция Администраций почты и электросвязи.

### В России установлено следующее соответствие:

* Первая квалификационная категория соответствует полной лицензии СЕРТ. Для её получения нужно сдать теоретический экзамен по программе полной лицензии СЕРТ (T/R 61-02) и дополнительно продемонстрировать умение принять передаваемый азбукой Морзе текст объёмом 250 знаков со скоростью 60 знаков в минуту;

* Вторая квалификационная категория соответствует полной лицензии СЕРТ. Для её получения нужно сдать теоретический экзамен по программе полной лицензии СЕРТ (T/R 61-02);

* Третья квалификационная категория соответствует лицензии СЕРТ новичка. Для её получения нужно сдать теоретический экзамен по программе лицензии СЕРТ новичка (ERC32).

* Четвёртая квалификационная категория не имеет соответствия какой – либо лицензии СЕРТ. Но для её получения нужно сдать теоретический экзамен по программе лицензии СЕРТ начального уровня (ECC 89). С ней можно взаимодейтсвовать только в УКВ диапазонах: ОВЧ-VHF, УВЧ-UHF, СВЧ-SHF и КВЧ-EHF.

Радиолюбитель, имеющий лицензию СЕРТ (полную или новичка (CEPT NOVICE), может свободно, не получая национальную лицензию страны пребывания, осуществлять передачи в течение 90 дней из страны, присоединившейся к рекомендациям СЕРТ T/R61-01 (для полной лицензии СЕРТ) и сообщению ECC(05)06 (для лицензии СЕРТ новичка).

Некоторых страны приняли рекомендации СЕРТ частично. Для передачи из этих стран необходимо получить местные разрешительные документы. Разрешенные для радиолюбителей полосы частот, виды работы и допустимые мощности могут отличаться от страны к стране. TODO Изучить законодательство конкретных стран, в которые ты планируешь.

# Инфа по позывному
Международным союзом электросвязи (МСЭ) выделены Администрации связи Российской Федерации блоки международных серий **RAA-RZZ** и **UAA-UIZ** для назначения позывных сигналов средствам радиосвязи всех назначений.

В позывном символы должны стоять так: один знак и одна цифра или два знака и одна цифра, сопровождаемая группой не более, чем три знака.

Любительским радиостанциям индивидуального и коллективного пользования присваиваются шестисимвольные позывные сигналы. Радиостанциям 1-й категории присваиваются пятисимвольные.

# Техническая база 

Задачи всех приёмников - это:
* Усиление
* Фильтрация
* Демодуляция(т.е. извлечение инфи из сигнала, который приняля антена)

Совсем ранние приёмники были детекторными(не давали усиления за счёт источников питания. Сильно устарели, изучение практического смысла не имеет)

### Супергетеродинный приёмник(90% современных приёмников)

![](/radio/radioResourses/супергетеродинный_приёмник.png)

Структурная схема содержит следующие элементы: 
* **Bx.У** — входное устройство 
* **УВЧ** — усилитель высокой частоты. Усиливает слабый сигнал, принятый антенной, до уровня, необходимого для работы смесителя
* **СМ** - смеситель. Преобразует частоту сигнала, смешивая его с сигналом гетеродина(**Г**); на выходе получается промежуточная частота
* **УПЧ** - усилитель промежуточной частоты
* **Д** - детектор (демодулятор). Извлекает из высокочастотного сигнала низкочастотную (звуковую) информацию.
* **УНЧ** Усилитель низкой частоты
* Настройка - изменяет частоту гетеродина. Да, это та самая крутилка
p.s. поидее гетеродин во всём мире - это система из смесителя и того, что в россии называется гетеродином.

### Cпособы модуляции (упаковки) радиосигнала. 
Их делят по типу передаваемой информации: 
#### Аналоговые. Меняют параметры волны так, чтобы она повторяла колебания голоса/звука:

* AM (Amplitude Modulation / Амплитудная модуляция) - Применяется для передачи речи и музыки (обычное УКВ/СВ-радиовещание). Сигнал меняет свою мощность (амплитуду) в такт с голосом.

* SSB (Single Sideband / Однополосная модуляция) - усовершенствованная версия для далней голосовой связи(морская, авиационная, любительская радиосвязь). Она экономит мощность, передавая только часть сигнала, но сохраняя голос.

* FM (Frequency Modulation / Частотная модуляция) - 
передача речи и музыки с высоким качеством (FM-радиостанции на УКВ). Сигнал меняет свою частоту в такт с голосом.

#### Цифровые (для данных/текста): Включают и выключают сигнал или меняют частоту по строгим законам, кодируя буквы в наборы импульсов (как азбука Морзе, но автоматическая):
* RTTY (Radiotelotype / Радиотелетайп). Применяется для передачи текста. Буквы превращаются в двоичный код (обычно 5-битный код Бодо). Радиостанция передает эти «единицы и нули», меняя частоту сигнала (частотная манипуляция, FSK) или включая/выключая тон. Приемник декодирует это обратно в буквы.
    
### Типы фильтров в тракте промжуточной частоты для разных режимов работы

| Режим | Оптимальная полоса фильтра | Тип фильтра | Зачем это нужно? |
| :--- | :--- | :--- | :--- |
| **SSB (телефон)** | 2,4–3 кГц | Кварцевый | Передает только голосовой диапазон (речь), отсекая лишние шумы и соседние станции. |
| **CW (телеграф, азбука Морзе)** | 500 Гц (иногда 250 Гц) | Кварцевый, сильно узкополосный | Сигнал CW занимает очень узкую полосу. Узкий фильтр вырезает практически все помехи и шумы, оставляя только «писк» морзянки. Это позволяет слышать очень слабые сигналы в грязи помех. |
| **AM (Амплитудная модуляция)** | 6–9 кГц | Керамический или LC | AM-сигнал содержит несущую и две боковые полосы, поэтому ему нужна широкая полоса для сохранения качества звука. |
| **FM (Частотная модуляция)** | 15–25 кГц | Керамический (дискриминаторный) | Сигнал ЧМ широкий, и для него нужен широкий тракт, чтобы не исказить частотные отклонения. |
| **Цифровые виды (FT8, RTTY, PSK)** | 2,4–2,7 кГц (или до 3 кГц) | Кварцевый | Для цифры нужна полоса, сопоставимая с SSB, но важно, чтобы фильтр был линейным по фазе (без перекосов), иначе цифровые данные исказятся. |


### Антенно фидерный тракт:
В качестве линии питания антены нужно использовать коаксиновый каблель или двохполосную линия.

Коаксиальное сопротивление(вернее волновое сопротивление коаксиального кабеля) — это «пропускная способность» кабеля для ВЧ-энергии, которая зависит исключительно от геометрии кабеля и материала изолятора (диэлектрика) между жилами.

Вычисляется по формлле:
``Z = √(L / C)``, где ``L`` - индуктивность кабеля (зависит от диаметра и расстояния между проводниками), ``C`` - ёмкость между центральной жилой и оплёткой (зависит от диэлектрика).

| Значение | Где применяется | Почему именно так? |
| :--- | :--- | :--- |
| **50 Ом** | **Радиосвязь** (трансиверы, антенны, любительские радиостанции, Wi-Fi, рации). | Компромисс между максимальной передаваемой мощностью (оптимум ~30 Ом) и минимальными потерями в кабеле (оптимум ~75 Ом). 50 Ом — «золотая середина» для работы с мощными передатчиками. |
| **75 Ом** | **Телевидение** и **спутниковое ТВ** (антенны, тюнеры, видеонаблюдение). | Обеспечивает минимальные потери сигнала, что критично для приёма, а не для передачи. |

### Компрессор речевого сигнала(PROC, COMP)
КРС выравневает голос, снижая динамический диапазон речи. Его нужно включать, когда в микрофон попадает много постороннего шума и звуков разной громкоси или при часто меняющейся громкости голоса. 

Как это работает?

Он следит за уровнем голоса и автоматически изменяет коэфицент усиления усилителя микрофона. Слабые звуки усиливает сильнее, а сильные - слабее. Это делает среднюю мощность выше, но не выше пиковой. За счёт этого на дгугом конце лучше слышно.

### Радиоаврора
Радиоаврора - это отражение радиоволн от приполярных областей ионосферы во время магнитных бурь.

### Однополо́сная модуля́ция
Однополо́сная модуля́ция (амплиту́дная модуляция с одно́й боково́й полосо́й) — разновидность амплитудной модуляции (AM), широко применяемая в аппаратуре каналообразования для эффективного использования спектра канала и мощности передающей радиоаппаратуры. Использует вернхнюю боковую полосу.

# Система оценки слыишимости RS/RST
Система RS применяется для SSB, AM, FM.

Система RST применяется для телеграфа — CW

R - Readability (Разборчивость) от 1 до 5.
S - Strength (сила) - от 1 до 9 + (10-20).
T — Tone (тон) — оценка качества звука

**Разборчивость (R — Readability):**

| Оценка R | Что означает |
| :--- | :--- |
| **1** | Неразборчиво (сплошной шум, слов не понять) |
| **2** | Едва разборчиво (отдельные слова удаётся разобрать) |
| **3** | Разборчиво с трудом (но понимать можно) |
| **4** | Разборчиво хорошо (почти без помех) |
| **5** | Отлично, абсолютно разборчиво (без помех, как по телефону) |

**Сила сигнала (S — Strength):**

| Оценка S | Что означает |
| :--- | :--- |
| **1–3** | Очень слабый сигнал (едва слышно) |
| **4–5** | Умеренный сигнал (слышно, но тихо) |
| **6–7** | Хороший сигнал (громко) |
| **8** | Очень хороший (сильный) |
| **9** | Максимально сильный (S-метр на 9) |
| **9+ (10–20)** | Сигнал сильнее максимума (индикатор зашкаливает) |

**Тон (T — Tone):**

| Оценка T | Что означает |
| :--- | :--- |
| **1** | Очень плохой тон (гулкий, искажённый) |
| **5** | Средний тон (слышен «песок») |
| **9** | Идеальный, чистый тон (свист без шума и модуляции) |

**Пример в эфире:**
- Для SSB: «Мне слышно Вас пятьдесят девять (59)»
- Для CW: «Ваш сигнал пять-девять-девять (599)»
